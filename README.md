# Avante Tech - Interface Web
Projeto de interface web desenvolvido com HTML e CSS, com foco na criação de telas modernas, organizadas e funcionais para um sistema de organização de documentos para a empresa Avante Tech Jr.

## Sobre o projeto
O projeto Avante Carreiras consiste no desenvolvimento de um sistema web voltado para o setor de Recursos Humanos da Avante Tech Jr., com o objetivo de organizar, centralizar e automatizar a gestão de documentos dos membros.
Atualmente, o processo é realizado de forma manual utilizando Google Drive, planilhas e aplicativos de mensagens, o que gera retrabalho, risco de perda de dados e dificuldade de acesso às informações.
A proposta do sistema é permitir busca rápida por documentos, classificação automática, envio direto de arquivos e notificações inteligentes, tornando o processo mais eficiente, seguro e padronizado.
